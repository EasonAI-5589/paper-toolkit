## 核心风格特点

- **手绘/卡通风格**：线条不规则、有手绘感
- **简洁配色**：2-4 种主色，避免复杂渐变
- **清晰标注**：模块名称、箭头方向明确
- **适用场景**：LLM 架构图、Multi-Agent 流程图、综述框架图

## 漫画风 Prompt 模板

### 通用模板

```
请生成一张漫画风格的科研框架图，要求：
- 手绘线条风格，略带不规则感
- 配色简洁（3-4种主色）
- 模块用圆角矩形或手绘方框
- 箭头用手绘风格
- 背景纯白或浅色
- 标注清晰，字体简洁
```

### LLM 架构图模板

```
绘制一张卡通手绘风格的LLM架构图：
- 输入层 → Encoder → Attention → Decoder → 输出
- 每个模块用不同颜色的手绘方框
- 箭头用弯曲的手绘线条
- 整体风格轻松活泼，适合学术展示
- 不要写字，只画结构
```

### Multi-Agent 流程图模板

```
生成手绘漫画风格的Multi-Agent系统图：
- 多个Agent用拟人化图标表示
- Agent之间的通信用虚线箭头
- 中央有协调器/环境模块
- 配色温暖，有亲和力
- 适合论文Figure使用
```

## 绘制流程

```
1. 确定内容结构
   └── 列出所有模块和关系

2. 草图规划
   └── 纸上/iPad 画布局草图

3. AI 生成初版
   └── 用 Nano Banana / Gemini 生成

4. 精修调整
   └── PPT/Figma 调整细节
   └── 统一配色、字体

5. 导出
   └── PDF/SVG 用于 LaTeX
   └── PNG 用于 PPT/Word
```

## 常见问题

### Q: AI 生成的图有水印怎么办？
A:
1. 放大裁剪边缘
2. PS 中用蒙版/修复画笔遮盖
3. 重新生成时指定"无水印"

### Q: 手绘风格太随意，不够学术？
A:
1. 保持线条整洁，只是略带手绘感
2. 配色使用学术期刊常用色
3. 标注使用正式字体（Arial, Helvetica）

### Q: 如何保持多张图风格一致？
A:
1. 使用相同的 Prompt 模板
2. 固定配色方案
3. 统一线条粗细和圆角弧度
