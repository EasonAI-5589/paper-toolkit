## 绘图技巧总结

### 从莱昂系列学到的核心原则

1. **配色统一**
   - 每张图 3-4 种主色
   - 同系列论文保持风格一致
   - 附带色卡方便复用

2. **层次分明**
   - 输入 → 处理 → 输出 清晰
   - 模块边界用框线或背景色区分
   - 箭头方向一致（通常从左到右或从上到下）

3. **元素对齐**
   - 水平/垂直对齐
   - 间距均匀
   - 视觉重心平衡

4. **标注清晰**
   - 关键模块有名称
   - 数据流有箭头说明
   - 数学符号规范

### 常用配色方案

**渐变色系**
```
主色: #667eea → #764ba2 (蓝紫渐变)
辅色: #f093fb → #f5576c (粉紫渐变)
背景: #ffffff
```

**粉色系**
```
主色: #ff9a9e, #fecfef
辅色: #a18cd1, #fbc2eb
强调: #667eea
背景: #fafafa
```

**简洁风**
```
主色: #4facfe (科技蓝)
辅色: #00f2fe (青色)
强调: #43e97b (绿色)
背景: #ffffff
线条: #333333
```
