## ACL/LLM Conference Style Guide

For ACL, NeurIPS, ICML style figures:

### Key Principles
1. **Clean background** - White or very light gray
2. **Consistent colors** - Use 3-5 colors max, stick to a palette
3. **Clear labels** - Large font (12pt+), no clutter
4. **Vector format** - Export as PDF for LaTeX

### Recommended Workflow
1. **Sketch idea** → Paper/iPad
2. **Generate with AI** → Nano Banana Pro / Gemini 3
3. **Refine in PPT/Figma** → Add labels, adjust colors
4. **Export** → PDF/SVG for LaTeX

### Common Figure Types in LLM Papers
- **Architecture diagrams** - Transformer blocks, attention flow
- **Pipeline diagrams** - Training → Fine-tuning → Inference
- **Comparison charts** - Model performance tables
- **Visualization** - Attention heatmaps, embeddings
